# My Personal Portfolio SPA 🚀

Це навчальний проєкт **Single Page Application (SPA)**, створений за допомогою бібліотеки **React**. Проєкт презентує мене як розробника-початківця, містить інформацію про мої навички, поточні проєкти та плани на майбутнє.

## 📋 Опис проєкту
Додаток складається з трьох основних розділів, між якими реалізована миттєва навігація без перезавантаження сторінки:

1.  **Про мене (/about)** — детальна інформація про мої цінності, досвід та структурований список репозиторіїв (HTML/CSS, JS, React).
2.  **Моє місто (/my-city)** — інформація про Київ та інтеграція з зовнішнім API для отримання актуальних погодних даних у реальному часі.
3.  **Мій розвиток (/my-future)** — професійні цілі, кроки до їх досягнення та мотиваційний блок.

## 🛠 Технологічний стек
* **React 18** — основа додатку.
* **React Router v6** — організація клієнтської маршрутизації.
* **Hooks (useState, useEffect)** — керування станом та життєвим циклом компонентів.
* **Open-Meteo API** — отримання географічних та погодних даних.
* **CSS-in-JS & Custom CSS** — стилізація компонентів для створення сучасного UI.
